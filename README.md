# Tic Tac Toe Game

A modern, TypeScript-based tic-tac-toe game built with React and Tailwind CSS.

## Features

- Interactive 3x3 grid
- Two-player gameplay (X and O)
- Win detection for all combinations
- Draw detection
- Game statistics tracking
- Responsive design
- Clean, modern UI
